# Emoticon Picker

Easily insert emoticons into your notes using a **Command Palette**, **Hotkeys**, or **Ribbon Icon**.

![Emoticon Picker Screenshot](https://github.com/user-attachments/assets/2ba64383-576d-4657-adfd-06d20024af3c)

## Key Features

- 200+ built-in emoticons
- Add your own **custom emoticons**
- **Favourite** emoticons for quick access
- **Hide** emoticons you don’t use

## Installation

> **Status:** Not yet available on Obsidian Community Plugins

### Manual Installation

1. Navigate to your Vault: `.obsidian/plugins/`
2. Create a folder called `emoticon-picker`
3. Copy the following files into that folder:

   - `main.js`
   - `styles.css`
   - `emotions.json`
   - `manifest.json`
4. Reload Obsidian and enable the plugin from **Settings → Community Plugins**
