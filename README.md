# Emoticon Picker

Easily insert emoticons into your notes using a **Command Palette**, **Hotkeys**, or **Ribbon Icon**.

![Emoticon Picker Screenshot](https://github.com/user-attachments/assets/6cd5bef9-b6ee-46b2-961f-8a37f31dbdbe)

## Key Features

- 200+ built-in emoticons 
- Add your own **custom emoticons** 
- **Favourite** emoticons for quick access 
- **Hide** emoticons you don’t use 

## Installation
You can find it in Obsidian's Community Plugins, [here](https://community.obsidian.md/plugins/emoticon-picker)!

Report issues and request features [here](https://github.com/Kazi-Aidah/emoticon-picker/issues).
